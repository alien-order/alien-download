# 대용량 파일 다운로더

12만개 규모의 모델코드 파일을 안전하고 효율적으로 다운로드하는 시스템입니다.

## 기능

- 엑셀 파일에서 모델코드/URL 읽기
- 실시간 진행률 표시 (완료/실패 개수)
- 일시정지/재개/중지 기능
- 자동 결과 로깅 (엑셀 파일)
- 배치 처리 (1000개씩)
- 중간 결과 자동 저장

## 두 가지 버전

### 1. DB 조회 버전 (`run_db_downloader.py`)
- 엑셀에서 모델코드만 읽어서 Oracle DB에서 URL 조회
- 1000개씩 배치로 DB 쿼리
- DB 연결 정보 설정 필요

### 2. URL 미리 추출 버전 (`run_url_downloader.py`)  
- 엑셀에 모델코드와 URL이 모두 포함된 경우
- 컬럼 위치 설정 가능 (A, B, C... 또는 0, 1, 2...)
- DB 연결 불필요

## 설치 및 실행

```bash
pip install -r requirements.txt

# DB 조회 버전 실행
python run_db_downloader.py

# URL 버전 실행  
python run_url_downloader.py
```

## 사용법

1. **엑셀 파일 선택**: 모델코드가 포함된 엑셀 파일 선택
2. **시작 행 설정**: 처리를 시작할 행 번호 (1부터 시작)
3. **다운로드 폴더**: 파일이 저장될 폴더 지정
4. **DB 설정** (DB 버전만): Oracle DB 연결 정보 입력 및 테스트
5. **컬럼 설정** (URL 버전만): 모델코드와 URL 컬럼 위치 지정
6. **시작**: 다운로드 시작

## 진행 상태

- 실시간 진행률: `완료(133/12000), 실패(2/12000)` 형태로 표시
- 상세 로그: 각 파일의 다운로드 결과 실시간 표시
- 진행률 바: 전체 진행률 시각적 표시

## 결과 파일

- `download_result_YYYYMMDD_HHMMSS.xlsx`: 최종 결과
- `download_result_temp_YYYYMMDD_HHMMSS.xlsx`: 중간 저장 (100개마다)

### 결과 파일 구조
| 모델코드 | URL | 파일명 | 상태 | 메시지 | 처리시간 |
|---------|-----|--------|------|--------|----------|
| MODEL001 | http://... | MODEL001.jpg | O | 성공 | 2024-01-01 10:30:15 |
| MODEL002 | http://... | MODEL002.png | X | 네트워크 오류 | 2024-01-01 10:30:20 |

## 주의사항

- 대용량 처리를 위해 배치 단위로 처리
- 네트워크 오류 시 자동 재시도 없음 (결과에서 확인 후 수동 처리)
- DB 버전 사용 시 실제 테이블명과 컬럼명 수정 필요
- 중지 시 결과 파일이 자동 저장됨
